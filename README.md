# 📊 Crypto Trader Sentiment Dashboard

An interactive Streamlit dashboard that analyzes crypto trader performance against Fear & Greed market sentiment.

This project combines trading analytics, sentiment analysis, and data visualization to identify how market psychology impacts profitability and trading outcomes.

# 📌 Features

- Interactive Streamlit dashboard
- KPI metrics and trading insights
- Profit analysis by market sentiment
- Coin-wise profitability analysis
- Monthly performance trends
- Risk analysis using standard deviation
- Download filtered dataset
- Dynamic filtering options

# 🛠️ Tech Stack

- Python
- Pandas
- NumPy
- Matplotlib
- Streamlit

---

# 📂 Dataset

The project uses:

- Historical crypto trading data
- Fear & Greed sentiment index data

---

# 📈 Key Insights

- Trader profitability changes across different market sentiments
- Certain coins perform better during greed-driven markets
- High-profit periods may also carry higher volatility and risk
- Market psychology can significantly impact trading outcomes

# ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/crypto-dashboard.git
```

Move into the project folder:

```bash
cd crypto-dashboard
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the Streamlit app:

```bash
streamlit run app.py
```
